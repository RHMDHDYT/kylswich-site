# KylSwich Site

Interactive terminal-style homepage for KylSwich Studio, built with [Hugo](https://gohugo.io/) and the [hugo-theme-shell](https://github.com/Yukuro/hugo-theme-shell).

## Features

- Typewriter animation on page load
- Interactive terminal prompt with commands:
  - `projects` — list projects tree
  - `posts` — list recent posts
  - `contact` — get in touch
  - `clear` — clear terminal output

## Setup

```bash
# Clone with submodules
git clone --recurse-submodules https://github.com/RHMDHDYT/kylswich-site.git

# Run dev server
hugo server
```

## Deploy

```bash
hugo
```

Output goes to `public/`.
